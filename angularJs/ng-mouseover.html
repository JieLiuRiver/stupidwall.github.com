 /*  	ng-mouseover && ng-mouseup &&  ng-mousemove
 /*===============================================*/
	
	<div ng-mouseover="count = count + 1" ng-init="count=0">鼠标移动到我这!</div>
	<h1>{{count}}</h1>
	

	<div ng-mousemove="count = count + 1" ng-init="count=0">鼠标移动到我这!</div>
	<h1>{{count}}</h1>

	

				
			
				
	代码说明：
	ng-mouseover 指令用于告诉 AngularJS 鼠标移动到指定的 HTML 元素上时要执行的操作。
	ng-mousemove 指令用于告诉 AngularJS 鼠标在 HTML 元素上移动时要执行的操作。
	ng-mouseup规定当在元素上松开鼠标按钮时的行为
