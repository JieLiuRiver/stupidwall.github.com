 /*  	ng-keydown && ng-keypress && ng-keyup
 /*===============================================*/
	<input ng-focus="count = count + 1" ng-init="count=0" />
	<h1>{{count}}</h1>
	
	
	<input ng-keypress="count = count + 1" ng-init="count=0" />
	<h1>{{count}}</h1>
	
	<input ng-keyup="count = count + 1" ng-init="count=0" />
	<h1>{{count}}</h1>

				
			
				
	代码说明：
	ng-keydown 指令用于告诉 AngularJS 在指定 HTML 元素上按下按键时需要的操作。
	ng-keydown 指令不会覆盖元素的原生 onkeydown 事件,
	ng-keypress 指令用于告诉 AngularJS 在指定 HTML 元素上按下按键时需要的操作。
	ng-keyup 指令用于告诉 AngularJS 在指定 HTML 元素上按键松开时需要的操作。
	 事件触发时，ng-keydown 表达式与原生的 onkeydown 事件将都会执行。
	按键敲击的事件顺序:
		Keydown
		Keypress
		Keyup
