/*===============================================*/
 /*  	AngularJs 常用工具
 /*===============================================*/
	1）、angular.lowercase()  将字符串转换为小写
	2）、angular.uppercase()  将字符串转换为大写
	3）、angular.copy()  数组或对象深度拷贝
	4）、angular.forEach()  对象或数组的迭代函数
	5）、angular.isArray()  如果引用的是数组返回 true
	6）、angular.isDate()  如果引用的是日期返回 true
	7）、angular.isDefined()  如果引用的已定义返回 true
	8）、angular.isElement() 如果引用的是 DOM 元素返回 true
	9）、angular.isFunction() 如果引用的是函数返回 true
	10）、angular.isNumber() 如果引用的是数字返回 true
	11）、angular.isObject() 如果引用的是对象返回 true
	12）、angular.isString() 如果引用的是字符串返回 true
	13）、angular.isUndefined() 如果引用的未定义返回 true
	14）、angular.equals() 如果两个对象相等返回 true
	15）、angular.fromJSON() 反序列化 JSON 字符串
	16）、angular.toJSON() 序列化 JSON 字符串
	17）、angular.bootstrap() 手动启动 AngularJS
	18）、angular.element() 包裹着一部分DOM element或者是HTML字符串，
	把它作为一个jQuery元素来处理。
