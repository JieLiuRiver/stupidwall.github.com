								指的是：组合优化的问题，多项复杂程度的非确定性问题。
								
								1、通过动态规划算法，求解最优性质的问题
								2、所有可能性计算出来
								3、分析： 
									1、当什么都没有的时候，最优就是0
									2、当背包容量2，而仓库里有的是3、4; 最优还是0， 放不下。
									3、当背包容量5，仓库是有3、4,最优还是4.
									4、可以放2个物品的时候：
										
                                js:
								var M = 5; //我们又五件商品
                                var W = 16;  //我们有个包，16的容器。
                                var arrP = [4,5,10,11,12];  //五件物品中，每一件的价值
                                var arrW  = [3,4,5,7,8];   //每件物品占的容器
                                //包里怎么装， 能得到最大的价值？
                                
                                function show( M, W, arrP, arrW ){
                                		var result = [];
                                        for( var i=0; i<=M; i++ ){
                                        	result[i] = [];
                                            for( var k=0; k<=W; k++ ){
                                            		if( i == 0 ){
                                                    		result[i][k] = 0;
                                                    }else if( arrW[i-1]  > k){
                                                    	retsult[i][k]  = result[i-1][k];
                                                    }else{
                                                    	result[i][k] = Math.max( arrP[i-1] +  result[i-1][k-arrW[i-1], result[i-1][k] )
                                                    }
                                            }
                                        }
                                        return result[i-1][k-1];
                                }
