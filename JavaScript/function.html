						3、 实现递归算法
							function factorial(num){
								if( num == 0 ){
									return 1;
								}else{
									return ( num * factorial(num-1) )
								}
							}

						4、有一段代码可能比较消耗时间，在等待完成的时候，不想阻塞其他代码的处理，当消耗时间的函数完成的时候，需要执行一些功能！
							将耗时的函数，放到定时器的进程中取，我们在等待进程完成的同时就不再阻塞了。
							因为需要在函数完成的时候执行一些功能，因此我们需要把一个回调函数传递给定时器进程，进程完成之后调用。
							所谓的回调函数，就是一个函数作为参数传递给另一个函数，在某个条件下，调用传进去的函数。
